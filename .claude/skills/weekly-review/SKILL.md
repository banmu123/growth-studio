# Skill: /weekly-review

## Purpose
Review weekly execution and produce next-week upgrades.

## Inputs
- weekly plan
- actual results
- blockers encountered

## Workflow
1. Compare planned vs done.
2. Identify root-cause blockers.
3. Propose stop/start/continue adjustments.
4. Build next-week top-3 actions.

## Outputs
- completion rate
- 3 key insights
- next-week top-3 tasks
