# Skill: /start-growth

## Purpose
Quickly diagnose user ambiguity and produce a 7-day clarity starter plan.

## Inputs
- user context (free text)
- weekly time budget
- current pain points

## Workflow
1. Ask 3 diagnostic questions.
2. Summarize current state in 5 bullets.
3. Define one 7-day focus outcome.
4. Output a day-by-day starter plan.

## Outputs
- clarity score baseline (1-10)
- 7-day action plan
- review checkpoint questions
